鄭元喬申請書

個人簡介

您好，我叫做鄭元喬，在寄出通知的的當下為 Ruby on Rails 的工程師，在第一階端時就會進入待業狀態，之前也是 camp 的學員，大學畢業於日文系，為非本科生。

為什麼想成為前端、後端、或全端工程師？

大學科系在台灣的求職環境中較為劣勢，本身任職過雲端遊戲服務公司因而產生對寫程式工作的興趣，現實層面的考量是軟體工程師的薪資下限普遍比台灣其他產業高，未來被 AI 取代的機率也偏低。

為了成為軟體工程師，曾經做過什麼努力？

在去年疫情期間，我知道完全程式新手在只有三個月培訓下是很難轉職成功的，所以直接在 camp 開始前就離職全心投入學習，報名 camp 並完成小組專案且每天花 8 小時以上的學習課程，與製作小組專案。找到工作後也本個月也會盡量越讀程式學習書籍，例如 0 陷阱！0 誤解！8 天重新認識 JavaScript！

如果參與這個計畫，會怎麼安排學習時間？

會依照之前上班的慣例於早上 8:30 起床，之後進入學習狀態。課程安排前幾週會以語法為主。每日花費時間估算為 8 個小時左右。如果遇到特別棘手的問題就會再把時間往上加，會盡量把每週任務拆成每日進度，並且執行。

如果參與這個計畫，預期會碰到什麼困難？你打算怎麼解決它？

我會在熟悉 python 語法使用與前端畫面切版上遇到一些挑戰，解決方法上我主要會以先 googel 找出可能的解法，之後實作，如果還是無法解決的話再整理成有順序的提問向助教們請教。

是否有想要加入的軟體公司？為什麼想加入該公司？

在台灣方面我希望能加入有 menter 制度的公司，並且有 code review 制度的公司，產業上為遊戲產業，例如 Garena，因為本人喜歡玩遊戲的關係，希望接觸到的產業多少與遊戲相關。並且希望未來技術提升到一定的程度在前往日本就職。

想要對我們說的事情？

請問助教的諮詢有分時段嗎？還是隨時在網路上都可以進行提問？未來在資料結構與演算法的學習內容，是會有固定教材還是不固定的提供內容呢？謝謝。
